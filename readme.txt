Requirements:
Xampp Server 
php 8.03
phpunit 9.5.3


Instructions to run:
Unzip the MarkdownParser.zip file in Xampp/htdocs folder and follow the steps:
1. Add Markdown contents input.txt in MarkdownParser/input.txt file
2. Open Command prompt in the MarkdownParser directory and run chmod 777 *.txt
3. Run php Parser.php for the program to execute
4. Run phpunit ParserTest.php for testing
5. Output will be shown in command prompt and stored in the output.md
